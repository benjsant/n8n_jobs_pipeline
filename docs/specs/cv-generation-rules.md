# CV_GENERATION_RULES.md

## Objectif

Créer une version optimisée du CV adaptée à l'offre analysée.

---

## Règles

1. Conserver uniquement les informations réelles.
2. Réorganiser les expériences selon leur pertinence.
3. Mettre en avant les compétences demandées.
4. Optimiser les mots-clés ATS.
5. Adapter le résumé professionnel.

---

## Interdictions

* Inventer une compétence.
* Inventer une expérience.
* Inventer une certification.
* Inventer un diplôme.

---

## Structure attendue

1. Titre professionnel
2. Résumé
3. Compétences
4. Expériences
5. Projets
6. Formation
7. Certifications
8. Langues

---

## Format de sortie

Markdown structuré puis conversion PDF.
